HTML

Расмотрим несколько элементов кода html.

Мы видим несколько строк кода, как думаете, какую команду они выполняют?


    <p class="my-style">
      Hello World!
    </p>

СSS

Теперь перед нами код CSS. Какую же команду выполняет этот элемент?

    
    body {
      font-family: 'Lato', arial, sans-serif;
      color: #444444;
      font-size: 16px;
      -webkit-font-smoothing: antialiased;
      -moz-osx-font-smoothing: grayscale;
    }
